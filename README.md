# 러스트 전사

![](rust_warrior.png)

요즘 특히 블록체인쪽에서 인기가 많은 러스트를 배우고 싶다는 주변 의견이 있고,
마침 러스트 프로그래밍을 시작해서 느슨하고 비동기적인 러스트 모임을 만듭니다.
따로 모이지 않고 각자 읽어보고 궁금한 점은 이슈로 이야기 나눌까 합니다.
러스트를 사용하지 않더라도 교양삼아 살펴보면 좋습니다.
